初始化
git init

用户名、邮箱
git config --global user.name '***'
git config --global user.email '****@***'

查看状态
git status
工作区-》暂存区
git add  文件名
暂存区-》分支
git commit -m '备注'

查看文件差异
git diff

查看日志
git log

查看所有日志
git reflog

回退上一个版本
git reset --hard HEAD^
回退指定版本
git reset --hard 版本号
从暂存区回退到工作区
git rm --cached 文件名
放弃修改，回退到上一个版本
git restore <file>

删除指定文件版本库
1：工作区删除指定文件 ；
2：git add .
     git commit -m

分支：
新建
git branch 分支名
查看
git branch
切换
git checkout 分支名
创建并切换
git checkout -b 分支名
删除
git branch -d 分支名
合并
git merge 分支名 

在项目中git操作流程：

在自己的分支上

git add .  // 添加文件到暂存区

git commit -m’备注’  //暂存区文件提交到自己的分支

git checkout master //切换到master 分支

git merge 自己的分支  //合并分支（有冲突解决冲突）

git pull   //拉取

git push   //推送

git checkout 自己的分支 //切换到自己的分支

git merge master //合并master分支内容
